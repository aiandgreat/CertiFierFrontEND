CERTIFIER is a web-based platform that automates the creation, verification,
and secure storage of digital certificates at UA CIT. It uses Django and PostgreSQL to
replace traditional paper processes which makes the administration more efficient and
cuts the department’s carbon footprint by a large amount. The EdDSA algorithm is
implemented in this project to make sure that the documents are safe and secure. It
gives students and faculty a way to manage real credentials that can’t be tampered with
and can be easily checked.
